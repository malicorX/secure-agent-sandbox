# Roadmap

This roadmap mirrors the 6-month milestone plan.

## M1 — Repo + runnable demo
- package skeleton
- deterministic seed runner scaffold
- structured logging format (JSONL)

## M2 — Task DSL + first task suite
- task definition format
- 6–10 tasks (exploration + coordination)
- baseline agents for sanity checks

## M3 — Simulated tools + policy engine
- capability-based tool API (simulated by default)
- violation metrics

## M4 — Evaluation harness + replay
- CLI runner
- replay artifacts
- determinism regression tests

## M5 — Optional hardened sandbox runner
- isolated tool execution mode (off by default)
- policy enforcement + logging parity with simulated tools

## M6 — Docs + v1.0
- stable API docs
- examples + adapters
- tagged v1.0 release
